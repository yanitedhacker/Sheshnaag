# Support

Project Sheshnaag is community-supported open source software.

## Where To Ask

- Bugs: open a GitHub issue with reproduction steps.
- Feature requests: open a GitHub issue with the workflow and expected outcome.
- Security reports: follow `SECURITY.md`.
- General usage questions: open a discussion or issue with environment details.

## Useful Context

Include the commit SHA, OS, Python version, Node version, deployment profile, and relevant command output. For maintainer assessments, include sanitized SBOM/VEX snippets when possible.

## Out Of Scope

Maintainers cannot help with unauthorized testing, exploit deployment, credential collection, phishing, target discovery, or unsafe publication of weaponized exploit material.
